## üìê Usage Guidelines

### Minimum Sizes
- **Horizontal Logo**: 120px width minimum
- **Stacked Logo**: 80px width minimum  
- **Icon Only**: 24px height minimum
- **Print**: 1 inch width minimum

### Clear Space
- Maintain clear space equal to the height of the "D" in DeCrown
- No other elements should appear within this protected area
- Applies to all sides of the logo

### Placement Rules
- Always use on backgrounds with sufficient contrast
- Never place on busy or distracting backgrounds
- Ensure readability at all sizes
- Maintain aspect ratio - never stretch or distort

## üìÅ File Organization

### File Naming Convention
```
decrown-logo-[variant]-[color]-[size].[format]

Examples:
- decrown-logo-horizontal-fullcolor-large.svg
- decrown-logo-stacked-monochrome-medium.png
- decrown-logo-icon-reverse-small.svg
```

### Required Formats
- **SVG**: Vector format for web and scalable applications
- **PNG**: High-resolution with transparent background
- **JPG**: For backgrounds that require solid colors
- **PDF**: For print applications

### Size Variations
- **Small**: 24px, 32px, 48px (icons)
- **Medium**: 64px, 96px, 128px (mobile apps)
- **Large**: 256px, 512px, 1024px (web headers)
- **Print**: 300 DPI versions for physical materials
